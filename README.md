# Loaner Request - ServiceNow

## Overview

The **Loaner Request Application** is a ServiceNow-based application designed to manage and automate the process of requesting and tracking loaner devices such as laptops, mobile phones, and other IT equipment.

This application helps organizations streamline device allocation, approvals, fulfillment, and return processes using ServiceNow workflows and catalog management.

---

## Features

* Loaner device request submission
* Automated approval workflow
* Asset tracking and management
* Request status monitoring
* Assignment and fulfillment handling
* Return tracking for borrowed devices
* Service Catalog integration
* Workflow automation using ServiceNow

---

## Technologies Used

* ServiceNow Platform
* Service Catalog
* Flow Designer / Workflow
* JavaScript (Client Scripts & Business Rules)
* Glide API
* Tables & Forms Configuration

---

## Application Workflow

1. User submits a loaner device request
2. Request is sent for approval
3. IT team fulfills the request
4. Device is assigned to the requester
5. Return process is tracked after usage

## Installation

1. Clone this repository:

```bash
git clone https://github.com/Karthik35k/Loaner-Request--ServiceNow.git
```

2. Import the update set/application into your ServiceNow instance.

3. Configure:

* Users
* Roles
* Assignment groups
* Catalog items
* Approval workflows

4. Test the application in your ServiceNow environment.
